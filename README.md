Aplikasi Warga Kelurahan merupakan proyek tugas mata kuliah Framework Programming yang dikembangkan oleh Irsal Fauzan Alfarizi (NIM: 23201268) di bawah bimbingan Bapak Achmad Noercholis, S.Kom., M.Kom.

Aplikasi ini dirancang untuk membantu proses administrasi dan pelayanan di tingkat kelurahan secara digital. Fitur-fitur utamanya meliputi:

Manajemen data warga (penduduk tetap, pendatang, pindah, meninggal)

Pembuatan surat keterangan online (domisili, usaha, kelakuan baik, dll)

Sistem login untuk admin dan petugas kelurahan

Dashboard statistik warga

CRUD data berbasis framework web (misalnya Django atau Laravel, tergantung framework yang kamu pakai)

Tujuan utama dari proyek ini adalah untuk mempermudah pihak kelurahan dalam melakukan pelayanan publik dengan sistem yang cepat, efisien, dan berbasis web.
